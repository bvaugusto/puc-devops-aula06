# puc-devops-aula06
API para realizar operações matemáticas

>operações:
- [:white_check_mark:] Soma de números
- [:white_check_mark:] Subtração de números
- [:white_check_mark:] Multiplicação de números
- [] Divisão de números

##Configuração
> npm install -g nodemon
> yarn

##Execução
> yarn start

##Documentação
>http://localhost:3000/api-docs
